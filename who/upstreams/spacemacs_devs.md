# Spacemacs Developers (collective)

## Role

Contributors who author and maintain Spacemacs layers we depend on. The roster shifts; names attached to specific layers appear in each layer's `README.org` after install.

## Layers we depend on

The canonical list lives in `what/standard/layers.md`. Phase 2 baseline:

- `lsp` — Language Server Protocol orchestration
- `python` — Python with ML emphasis
- `typescript` — TypeScript / JavaScript
- `rust` — Rust language layer
- `markdown` — Markdown editing + preview
- `org` — Org-mode
- `git` — Magit integration
- `treemacs` — File tree
- `helm` — Selection framework (alternative: `ivy`)
- `auto-completion` — Company-mode

The `adna` layer (in `what/standard/layers/adna/`) is Spacemacs.aDNA's own contribution — not from upstream.

## Engagement

When a layer needs improvement that would benefit other Spacemacs users, file the issue/PR upstream. Every upstream contribution gets recorded here with link to the PR.
