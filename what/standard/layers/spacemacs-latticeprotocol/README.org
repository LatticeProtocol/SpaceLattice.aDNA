* Description

The =spacemacs-latticeprotocol= layer is the LP distribution layer for
Spacemacs.aDNA. It extends the upstream =spacemacs= distribution with:

- The =latticeprotocol-theme= local package (dark + light variants)
- LP-specific branding overrides via =setq= (no core file patching)
- The =SPC o l= LP keybinding prefix (aDNA navigation + Claude Code IDE)

This layer follows the vault-only model established in ADR-024: all LP code
lives in =what/standard/layers/= inside the vault. No fork of
=syl20bnr/spacemacs= is maintained.

* Deployment

=skill_install= Step 5 symlinks this layer into =~/.emacs.d/private/layers/=:

#+begin_src bash
ln -s "$VAULT/what/standard/layers/spacemacs-latticeprotocol" \
      ~/.emacs.d/private/layers/spacemacs-latticeprotocol
#+end_src

In =dotfile.spacemacs.tmpl=, set:

#+begin_src emacs-lisp
dotspacemacs-distribution 'spacemacs-latticeprotocol
#+end_src
