* Description

The =adna= Spacemacs layer makes Emacs aware of aDNA vaults — directories
matching =*.aDNA/=. When a buffer's path has any =*.aDNA/= ancestor, the layer
activates =adna-mode=, parses the file's YAML frontmatter into buffer-local
variables, and surfaces the =SPC a= transient menu for triad-aware navigation.

This layer is the "wings" half of the Daedalus metaphor governing
Spacemacs.aDNA — tools that lift the operator above the labyrinth of
aDNA context-graph relations rather than just through it.

* Install

Activated automatically when this layer is listed in =dotspacemacs-configuration-layers=
of =~/.spacemacs=. =what/standard/dotfile.spacemacs.tmpl= includes =adna= in the
canonical layer list with the =adna-claude-code-command= variable exposed.

The layer's source lives at =<vault>/what/standard/layers/adna/=. =skill_install=
symlinks it to =~/.emacs.d/private/layers/adna/=.
